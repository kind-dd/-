Game Description

Monster Survivors 是一款充满挑战性的生存冒险游戏。在这个充满危险的世界中，你将扮演一位勇敢的幸存者，面对各种可怕的怪物和挑战。

游戏采用实时战斗系统，你需要运用策略和技巧来击败敌人。通过收集资源、升级装备和学习新技能，你的角色将变得更加强大。

游戏世界充满了随机事件和隐藏内容，每次游戏体验都会有所不同。与其他玩家竞争，看看谁能在怪物世界中生存得更久！

如何游戏

  • 使用 WASD 或方向键移动角色
  • 点击鼠标或按空格键进行攻击
  • 收集经验值来升级角色
  • 躲避危险，寻找安全的生存策略

System Requirements

最低配置

  • • 操作系统：Windows 7 或更高版本
  • • 处理器：Intel Core i3 或同等性能
  • • 内存：4 GB RAM
  • • 显卡：支持 WebGL 的显卡
  • • 网络：稳定的互联网连接

推荐配置

  • • 操作系统：Windows 10 或更高版本
  • • 处理器：Intel Core i5 或同等性能
  • • 内存：8 GB RAM
  • • 显卡：现代显卡，支持 WebGL 2.0
  • • 网络：高速互联网连接
